About

This website was made in the context of a university project for a course called Web Technologies. The goal of this project is to provide an initiation to web technologies and web development. To build this project, we used HTML, CSS and JavaScript (coupled with jquery) for the front-end and PHP (vanilla) for the back-end.
The goal of this project is to create a messaging app where users can add friends and chat with them in real time. Rather than trying to implement a ton of features, we focused on clean code and architecture, effectively leading to a safer and more reliable user experience.
